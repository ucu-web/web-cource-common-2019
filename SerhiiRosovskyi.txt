3. Can you explain the difference between coding a web site to be responsive versus using a mobile-first strategy?
Responsive - це коли ми починаємо верстку нашого сайту, базуючись на великому розміру екрану, а далі адаптивно це верстаємо для менших екранів.
Mobile-first - весь дизайн базується на забезпечинні досвіді юзера, який використовує телефн.
